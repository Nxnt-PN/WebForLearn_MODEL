คำอธิบายโมเดล Logistic Regression

โมเดลที่ใช้พยากรณ์ผลรอดชีวิตของ Titanic คือ Logistic Regression ซึ่งเป็นโมเดลสำหรับการจำแนกประเภทแบบสองกลุ่ม

1. หลักการของ Logistic Regression

Logistic Regression เป็นอัลกอริธึมที่ใช้จำแนกประเภทของข้อมูลที่มีผลลัพธ์เป็นสองค่า เช่น รอดชีวิต หรือ ไม่รอดชีวิต

โมเดลนี้ใช้ Sigmoid Function เพื่อแปลงค่าผลลัพธ์ให้อยู่ในช่วง 0 ถึง 1 และคำนวณโอกาสรอดชีวิตของผู้โดยสาร

\( P(Y=1 | X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1X_1 + ... + \beta_nX_n)}} \)

2. ข้อมูลที่ใช้ในโมเดล

โมเดลใช้ข้อมูลจาก Titanic Dataset ซึ่งมีตัวแปรสำคัญดังนี้:

คุณลักษณะ รายละเอียด
Pclass ชั้นโดยสาร (1, 2, 3)
Sex เพศ (0 = ชาย, 1 = หญิง)
Age อายุของผู้โดยสาร
SibSp จำนวนพี่น้อง/คู่สมรส
Parch จำนวนพ่อแม่/ลูก
Fare ค่าโดยสาร
Embarked ท่าเรือที่ขึ้นเรือ (0 = Cherbourg, 1 = Queenstown, 2 = Southampton)

3. ขั้นตอนการสร้างโมเดล

  1. โหลดข้อมูล: นำเข้าชุดข้อมูล Titanic
  2. ทำความสะอาดข้อมูล: จัดการค่าที่ขาดหายและแปลงข้อมูลให้อยู่ในรูปแบบที่ใช้งานได้
  3. แบ่งข้อมูล: แบ่งข้อมูลเป็นชุด Train (80%) และ Test (20%)
  4. ฝึกโมเดล: ใช้ Logistic Regression กับชุด Train
  5. ประเมินผล: ใช้ Accuracy, Precision, Recall และ F1-score

4. ผลลัพธ์และความแม่นยำของโมเดล

โมเดลสามารถพยากรณ์ผลรอดชีวิตได้อย่างแม่นยำ โดยมีค่าชี้วัดดังนี้:

  • Accuracy: 80%
  • Precision: 78%
  • Recall: 75%
  • F1-score: 76%
